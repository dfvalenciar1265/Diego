¿App nativa, PWA o Web?

Para tu caso: 90% móvil, equipo pequeño, MVP rápido

A

App Nativa (React Native / Flutter)

Una app real en App Store y Google Play. Se comporta exactamente como cualquier app de tu teléfono.

✅ Ventajas

  • Experiencia 100% nativa
  • Push notifications reales
  • Acceso completo a cámara/GPS
  • Funciona offline

⚠️ Desventajas

  • 3-4x más tiempo de desarrollo
  • Revisión App Store (días/semanas)
  • Updates lentos
  • Dos codebases (iOS + Android)
⏱️

Tiempo estimado de MVP

3–5 meses

B

PWA — Progressive Web App RECOMENDADO

Una web app que se instala en el teléfono como una app — ícono en pantalla de inicio, sin barra del navegador, push notifications, cámara. Sin App Store.

¿Cómo la instala tu equipo?

1. Abren el link
→
2. "Agregar a inicio"
→
3. ¡Listo! 🎉

✅ Ventajas

  • Se ve y siente como app
  • Un solo codebase
  • Deploy instantáneo
  • Sin App Store
  • Push notifications ✓
  • Cámara y GPS ✓
  • Funciona offline (básico)

⚠️ Límites

  • No está en App Store
  • Push en iOS requiere iOS 16.4+
  • Offline limitado vs nativa
⏱️

Tiempo estimado de MVP

4–6 semanas

C

Web Responsive (solo navegador)

Una página web adaptada a móvil. Se accede siempre desde el navegador, sin instalación.

✅ Ventajas

  • Más simple de construir
  • Cualquier dispositivo
  • Sin instalación

⚠️ Desventajas

  • Sin push notifications
  • Barra del navegador siempre visible
  • UX inferior en móvil
  • Sin acceso a cámara fácil
⏱️

Tiempo estimado de MVP

3–4 semanas
